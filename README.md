# 📊 Calculadora Acadêmica Univesp

Uma aplicação web desenvolvida para auxiliar estudantes na gestão de suas médias acadêmicas, automatizando o cálculo de notas com base nos critérios da instituição.

## 🚀 Tecnologias Utilizadas
* **Linguagem:** Python
* **Framework Web:** Streamlit
* **Versionamento:** Git
* **Deploy:** Streamlit Community Cloud & GitHub

## 🛠️ Funcionalidades
* Cálculo automatizado de médias finais.
* Interface responsiva e intuitiva para inserção de dados.
* Lógica de programação aplicada para validação de critérios de aprovação.

## 📂 Como executar o projeto
1. Clone o repositório: `git clone https://github.com/seu-usuario/calculadora-univesp`
2. Instale as dependências: `pip install streamlit`
3. Execute a aplicação: `streamlit run app.py`

## 🧠 Contexto Técnico e QA
Este projeto foi desenvolvido como parte do meu aprendizado em **Ciência de Dados**, aplicando conceitos de lógica de programação e arquitetura de software. Durante o desenvolvimento, foquei em **Garantia de Qualidade (QA)** através de testes manuais de fluxo para assegurar que os cálculos matemáticos fossem precisos para todos os cenários de notas.
